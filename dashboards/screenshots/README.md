# Dashboard Screenshots

This directory contains screenshots of all Looker dashboards for portfolio use.

## Screenshots

- `executive_summary.png` - Executive Summary dashboard
- `mrr_deep_dive.png` - MRR Deep Dive dashboard
- `churn_analysis.png` - Churn Analysis dashboard
- `cohort_retention.png` - Cohort Retention dashboard
- `unit_economics.png` - Unit Economics dashboard

## Taking Screenshots

### macOS
1. Open the dashboard in Looker
2. Press `Cmd + Shift + 4` to capture a selection
3. Select the dashboard area
4. Save to this directory

### Windows
1. Open the dashboard in Looker
2. Press `Windows + Shift + S` to open Snipping Tool
3. Select the dashboard area
4. Save to this directory

### Tips
- Use full-screen mode for cleaner screenshots
- Hide browser UI elements (F11 for full screen)
- Use high resolution (at least 1920x1080)
- Capture the entire dashboard view
- Ensure all tiles are loaded before capturing
